## Цель

Это не классический перцептрон. Это **когнитивный нейрон** — вычислительный примитив, спроектированный как строительный блок для искусственных когнитивных систем на пути к AGI.

Классические нейроны `y = σ(Σwx + b)` — это линейные классификаторы с нелинейностью. Они математически удобны, но биологически и когнитивно примитивны. Наш нейрон кардинально отличается.

## Архитектура

```
        АПИКАЛЬНЫЕ ДЕНДРИТЫ (top-down)
              │
              │ ← предсказания, контекст, внимание
              │
         ┌────┴────┐
         │  СОМА   │ ← интеграция + нелинейность + состояние
         └────┬────┘
              │
              │ ← сенсорный вход, латеральные связи
              │
        БАЗАЛЬНЫЕ ДЕНДРИТЫ (bottom-up)
```

### 1. Дендритное вычисление

Каждый нейрон имеет **два дендритных дерева**:
- **Базальные дендриты** (8 сегментов): обрабатывают входной/feedforward сигнал
- **Апикальные дендриты** (4 сегмента): обрабатывают контекст/предсказания сверху

Каждый сегмент — это **независимый детектор совпадений** с собственными весами, порогами и нелинейностью. Это даёт одному нейрону вычислительную мощность **маленькой сети**, а не одного линейного классификатора.

**Синаптическая перманентность**: каждый синапс имеет значение "консолидации" [0,1]. Только синапсы выше порога активны. Это даёт **структурную пластичность** — сеть может перестраиваться, наращивая и удаляя связи.

### 2. Временна́я динамика

Нейрон — это не функция, а **динамическая система**:
- **Мембранный потенциал**: утечка + интеграция входов
- **Адаптация**: частотная адаптация предотвращает runaway-активацию
- **Следы допустимости** (eligibility traces): запоминают "что произошло" для отложенного обучения

### 3. Предиктивное кодирование

Ядро когнитивной архитектуры. Нейрон **предсказывает** свой вход:

```
Ошибка предсказания = Базальный вход − Апикальное предсказание
```

Только **удивление** (prediction error) распространяется вверх по иерархии. Это фундаментальный принцип, лежащий в основе теории предиктивного мозга.

**Точность** (precision): ацетилхолин модулирует, насколько нейрон доверяет входному сигналу vs. предсказанию.

### 4. Режимы активации

Три качественно различных режима:

| Режим | Условие | Значение |
|-------|---------|----------|
| **SILENT** | V < θ_tonic | Накопление свидетельств |
| **TONIC** | θ_tonic ≤ V < θ_burst | Градуированный сигнал (локальное вычисление) |
| **BURST** | V ≥ θ_burst или высокая ошибка | Мощный сигнал (коммуникация + пластичность) |

Burst-режим вызывается неожиданностью — это механизм **внимания и алерта**.

### 5. Нейромодуляция

Глобальные нейромодуляторы меняют **режим работы** нейрона:

| Модулятор | Аналог | Эффект |
|-----------|--------|--------|
| **Дофамин** | Reward prediction error | Ворота пластичности, мотивация |
| **Ацетилхолин** | Неопределённость | Усиливает обучение, доверие к входу |
| **Норадреналин** | Новизна/ароusal | Усиление gain, эксплорация |
| **Серотонин** | Временной горизонт | Терпеливость, долгосрочное vs краткосрочное |

### 6. Мульти-правильная пластичность

Шесть правил обучения, работающих совместно:

1. **Хеббовское** с BCM-скользящим порогом — усиление ко-активных синапсов
2. **Предиктивное** — уменьшение ошибки предсказания (апикальные дендриты)
3. **Ошибочно-управляемое** — совместимое с backprop (когда есть target)
4. **Модулированное наградой** — трёхфакторное правило (Hebb × Reward)
5. **Структурная пластичность** — рост/обрезка синапсов через перманентность
6. **Мета-пластичность** — адаптация самих скоростей обучения

### 7. Гомеостаз

Нейрон поддерживает **стабильный режим работы**:
- Отслеживает среднюю активность
- Корректирует порог возбудимости
- Модулирует собственный gain
- Предотвращает "молчание" и "взрыв" активности

### 8. Мета-пластичность

Обучение адаптирует **само обучение**:
- Высокая удивлённость → увеличить скорость обучения
- Низкая удивлённость → уменьшить (консолидация)
- Реализует "synaptic tagging and capture"

## Сетевой уровень

### CognitiveLayer
Слой нейронов с:
- **Латеральное торможение**: активные нейроны подавляют соседей → разреженное кодирование
- **Рекуррентный контекст**: предыдущий выход подаётся обратно

### CognitiveCircuit
Многослойная иерархия с двунаправленным потоком:
```
     Слой 2 ←── предсказания ──→ Слой 2
       ↑ ошибки                       ↓ предсказания
     Слой 1 ←── предсказания ──→ Слой 1
       ↑ ошибки                       ↓ предсказания
     Слой 0 ←───── вход ─────────────────
```

## Направления развития для AGI

1. **Attention heads** на уровне дендритных сегментов
2. **Episodic memory** — быстрое запоминание конкретных паттернов  
3. **Working memory** — устойчивые аттракторы в рекуррентных контурах
4. **Continual learning** — предотвращение катастрофического забывания через перманентность
5. **Compositional representations** — связывание через синхронную burst-активность
6. **Causal reasoning** — interventional learning через нейромодуляцию
7. **Consciousness-like dynamics** — global workspace через burst-режим
8. **Self-modeling** — нейроны, моделирующие состояния других нейронов
